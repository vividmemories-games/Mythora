## Principles

1. **Systems over bespoke art** — reusable frames, rarity borders, skill icons
2. **Readability first** — board tiles and HP/resources beat illustration detail
3. **Dusk identity** — follow [Theme](../02_Design_System/Theme.md)
4. **AI-assisted production OK** — human review required before ship ([AB1 §11](AB1_Production_Standards.md#11-review-checklist-required-before-ship))
5. **Stable IDs & naming** — `hero_mage.png`, `enemy_goblin.png`, `tile_red.png` ([AB1 §6](AB1_Production_Standards.md#6-naming-conventions))
